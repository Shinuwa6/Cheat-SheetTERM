Git et son fonctionnement:

Git est utilisé pour versionner son code.

Commande de GIT via le terminal

ls Clone an existing repository
ls -all Create a new local repository
ls -lh Changed files in your working directory
cd "nom du rep" Changes to tracked files
rm "nom_du_fichier" Add all current changes in "files" to the next commit
mkdir "nom_du_rep" Add some changes in "files" to the next commit
rmdir "nom_du_repgit" Commit all local changes in tracked files
pwd Commit previously staged changes
cp "doc_source doc_destination" Change the last commit
cp -R "rep_source rep_destination" List all existing branches
mv "doc_source doc_destination" Switch head branch
$ top Create a new branch based on your current HEAD
$ who Create a new tracking branch based on a remote branch
$ w Delete a local branch
$ users Mark the current commit with a tag
$ write "nom_d'utilisateur" Show all commits, staring with newest
$ uptime Show changes over time for a specific file
